# 중간고사 간결 답안 (박스 삽입용)

각 문항 박스에 들어갈 간결한 풀이. Unicode 기반 (θ, Δ, ω, √, ², ³, ₀ 등).

---

## 1(a)
제1법칙: 물체에 작용하는 알짜힘이 0이면, 정지한 물체는 계속 정지, 운동하는 물체는 등속직선운동을 유지한다 (F_net=0 ⇒ v=일정).
관성 기준틀: 제1법칙이 성립하는 기준틀, 즉 가속하지 않는(등속으로 움직이거나 정지한) 기준틀.

## 1(b)
제2법칙 F_net = ma ⇒ a = F_net/m.
• 질량 일정, F → 2F:  a' = 2F/m = 2a.  가속도 2배.
• F 일정, m → 3m:  a' = F/(3m) = a/3.  가속도 1/3배.

## 1(c)
제3법칙: A가 B에 힘 F_AB 를 가하면 B도 A에 F_BA = −F_AB 를 가한다 (크기 같고 방향 반대, 서로 다른 두 물체에 작용).
책의 상호작용 쌍:
① 책–지구 (중력):  지구→책 = m·g (아래)   ↔   책→지구 = −m·g (위).
② 책–테이블 (수직항력):  테이블→책 = N (위)   ↔   책→테이블 = −N (아래).
주의: 책에 작용하는 중력(m·g)과 수직항력(N)은 둘 다 책에 작용하므로 작용-반작용 쌍이 아니라 힘의 평형이다.

## 2(a)
y(t) = v₀ sinθ₀·t − (1/2)g t².  y=0 ⇒ t_f = 2v₀ sinθ₀/g.
R = v₀ cosθ₀ · t_f = **v₀² sin(2θ₀)/g**

## 2(b)
(a)에서 t_f = **t(θ₀) = 2v₀ sinθ₀/g**

## 2(c)
sinθ₀ 가 최대일 때 비행시간 최대 ⇒ θ₀=90°에서 t_max = 2v₀/g.
t(θ₀)=0.5 t_max ⇒ 2v₀ sinθ₀/g = v₀/g ⇒ sinθ₀ = 1/2.
발사각 범위 (0°, 90°] 에서 **θ₀ = 30°**.

## 2(d)
v² = v_x² + v_y², v_x = v₀ cosθ₀ (일정), v_y(t) = v₀ sinθ₀ − g t.
v_x 가 일정하므로 v_y = 0 일 때 (궤적의 최고점) 속력이 최소. 이유: 중력은 연직 방향으로만 작용하여 v_x 는 보존되고 v_y 는 상승 중 감속→0→하강하며 증가.
θ₀=30°에서 **v_min = v₀ cos30° = (√3/2) v₀**.

## 3(a)
충격량-운동량 정리: **Δp = F Δt**.
초기 정지 ⇒ 최종 속도 **v = Δp/m = F Δt/m**.
일-운동에너지 정리: **W = ΔK = (1/2) m v² = F² (Δt)²/(2m)**.

## 3(b)
운동량 성분 보존 ⇔ 그 방향 알짜힘 = 0.
수평면 구간 (힘 F 작용 중):
• 수평: 알짜힘 = F ≠ 0 ⇒ 보존 ✗.
• 연직: 중력(−mg)+수직항력(+mg) = 0 ⇒ 보존 ✓.
경사면 구간 (중력 + 경사면 수직항력 N, 경사각 θ, N=mg cosθ):
• 수평: 알짜힘 = −N sinθ = −mg sinθ cosθ ≠ 0 ⇒ 보존 ✗.
• 연직: 알짜힘 = N cosθ − mg = −mg sin²θ ≠ 0 ⇒ 보존 ✗.
이유: 경사면의 수직항력이 수평·연직 양방향으로 성분을 가지기 때문.

## 3(c)
경사면 진입 직후: **K_i = (1/2) m v² = F²(Δt)²/(2m)**, **U_i = 0** (기준 높이).
정지 순간(경사 따라 거리 d, 높이 h = d sinθ): **K_f = 0**, **U_f = mg d sinθ**.
역학적 에너지 보존 (마찰 없음, 수직항력 일 하지 않음): (1/2) m v² = m g d sinθ.
⇒ **d = v²/(2 g sinθ) = F²(Δt)²/(2 m² g sinθ)**.
⇒ **U_f = F²(Δt)²/(2m)**.

## 4(a)
축에 대한 외부 토크 0 ⇒ **각운동량 보존**.
충돌 전: L_i = m v R (충격 매개변수 R).
충돌 후: I_f = (1/2) M R² + m R² = (M+2m) R²/2,  L_f = I_f ω.
mvR = (M+2m)R² ω/2  ⇒  **ω = 2mv / [(M+2m) R]**.

## 4(b)
K_i = (1/2) m v².
K_f = (1/2) I_f ω² = **m² v² / (M+2m)**.
K_f / K_i = 2m/(M+2m) < 1 (M>0).
⇒ 운동에너지가 감소하므로 **비탄성 충돌**. 입자가 원판에 붙으므로 **완전비탄성 충돌**.

## 4(c)
미는 힘은 반지름 방향 ⇒ 축에 대한 토크 0 ⇒ **각운동량 L = mvR 보존**.
r < R 일 때 관성모멘트: I(r) = (1/2) M R² + m r².
L = I(r) ω_f  ⇒  **ω_f = mvR / [(1/2) M R² + m r²] = 2 m v R / (M R² + 2 m r²)**.
