## 문제 1 (20점)

가속하는 엘리베이터 안에서의 겉보기 무게에 대해 답하시오.

**(a)** (5점) 질량 $m$인 사람이 체중계 위에 서서 엘리베이터를 타고 있다. 엘리베이터가 위로 가속도 $a$로 가속할 때, 사람에 대한 자유물체도를 그리시오.

**풀이:**

사람에게 작용하는 힘은 두 가지이다:

1. **중력** $\vec{W} = -mg\hat{j}$: 아래 방향
2. **수직항력(체중계의 읽힘)** $\vec{N} = N\hat{j}$: 위 방향 (체중계가 사람을 위로 미는 힘)

자유물체도:

```
       ↑ N (수직항력)
       |
      [m] (사람)
       |
       ↓ mg (중력)
```

$N$은 체중계가 읽는 겉보기 무게 $W_\text{app}$에 해당한다. (뉴턴의 제3법칙에 의해, 사람이 체중계를 누르는 힘과 체중계가 사람을 미는 힘은 크기가 같고 방향이 반대이므로, $W_\text{app} = N$이다.)

---

**(b)** (8점) 뉴턴의 제2법칙을 적용하여, 체중계가 읽는 겉보기 무게 $W_\text{app}$을 $m$, $g$, $a$로 나타내시오.

**풀이:**

위쪽을 양의 방향으로 정한다. 엘리베이터가 위로 가속도 $a$로 가속하므로, 사람도 같은 가속도로 운동한다.

뉴턴의 제2법칙 ($y$방향):

$$\sum F_y = ma$$

$$N - mg = ma$$

$$N = m(g + a)$$

체중계의 읽힘은 수직항력의 크기이므로:

$$\boxed{W_\text{app} = m(g + a)}$$

위로 가속할 때 ($a > 0$), 겉보기 무게는 실제 무게 $mg$보다 크다.

---

**(c)** (3점) 엘리베이터가 정지해 있거나 등속으로 움직일 때 겉보기 무게는 얼마인지 (b)의 결과를 확인하시오.

**풀이:**

정지 또는 등속 운동일 때 가속도 $a = 0$이다. (b)의 결과에 $a = 0$을 대입하면:

$$W_\text{app} = m(g + 0) = mg$$

따라서 체중계는 사람의 실제 무게 $mg$를 읽는다. 이는 예상과 일치한다.

---

**(d)** (4점) 엘리베이터의 줄이 끊어져 자유낙하하는 경우 ($a = -g$) 체중계의 읽힘과 그 의미를 논하시오.

**풀이:**

자유낙하 시 가속도는 $a = -g$ (아래 방향)이다. (b)의 결과에 대입하면:

$$W_\text{app} = m(g + (-g)) = m \cdot 0 = 0$$

체중계의 읽힘이 0이다. 이것이 **무중력 상태(weightlessness)** 이다.

물리적 의미: 사람과 엘리베이터(체중계 포함)가 동일한 가속도 $g$로 자유낙하하므로, 사람과 체중계 사이에 접촉력이 필요하지 않다. 중력 자체가 사라진 것이 아니라, 관측자(사람)와 바닥(체중계)이 함께 떨어지기 때문에 겉보기 무게가 0이 되는 것이다. 이는 국제우주정거장(ISS)에서 우주비행사가 경험하는 무중력과 동일한 원리이다.

## 문제 2 (25점)

일과 운동에너지에 대해 답하시오.

**(a)** (7점) 일-운동에너지 정리(work-kinetic energy theorem)를 서술하고, 질점에 작용하는 알짜힘 $F$와 뉴턴의 제2법칙으로부터 이를 유도하시오. (1차원)

**풀이:**

**서술:** 물체에 작용하는 알짜힘이 한 알짜일 $W_\text{net}$은 물체의 운동에너지 변화량과 같다.

$$W_\text{net} = \Delta K = K_f - K_i = \frac{1}{2}mv_f^2 - \frac{1}{2}mv_i^2$$

**유도:** 뉴턴의 제2법칙에서 출발한다:

$$F_\text{net} = ma$$

가속도를 속도와 위치로 표현한다. 연쇄법칙(chain rule)을 사용하면:

$$a = \frac{dv}{dt} = \frac{dv}{dx}\frac{dx}{dt} = v\frac{dv}{dx}$$

알짜힘이 한 일을 위치 $x_i$에서 $x_f$까지 적분한다:

$$W_\text{net} = \int_{x_i}^{x_f} F_\text{net} \, dx = \int_{x_i}^{x_f} ma \, dx = \int_{x_i}^{x_f} mv\frac{dv}{dx} \, dx$$

적분 변수를 $x$에서 $v$로 바꾸면:

$$W_\text{net} = \int_{v_i}^{v_f} mv \, dv = \frac{1}{2}mv^2 \Big|_{v_i}^{v_f} = \frac{1}{2}mv_f^2 - \frac{1}{2}mv_i^2$$

따라서:

$$\boxed{W_\text{net} = \frac{1}{2}mv_f^2 - \frac{1}{2}mv_i^2 = \Delta K}$$

---

**(b)** (8점) 질량 $m$인 물체에 위치에 따라 변하는 힘 $F(x) = F_0(1 - x/L)$이 작용한다 ($0 \le x \le L$). 물체가 $x = 0$에서 정지 상태로 출발할 때, $x = L$에서의 속력을 $F_0$, $L$, $m$으로 나타내시오.

**풀이:**

일-운동에너지 정리를 적용한다. 초기 운동에너지 $K_i = 0$ (정지 출발)이므로:

$$W = \Delta K = \frac{1}{2}mv^2 - 0 = \frac{1}{2}mv^2$$

힘 $F(x) = F_0(1 - x/L)$이 $x = 0$부터 $x = L$까지 한 일을 구한다:

$$W = \int_0^L F(x) \, dx = \int_0^L F_0\left(1 - \frac{x}{L}\right) dx$$

$$= F_0 \left[ x - \frac{x^2}{2L} \right]_0^L = F_0 \left( L - \frac{L^2}{2L} \right) = F_0 \left( L - \frac{L}{2} \right) = \frac{F_0 L}{2}$$

일-운동에너지 정리에서:

$$\frac{1}{2}mv^2 = \frac{F_0 L}{2}$$

$$v^2 = \frac{F_0 L}{m}$$

$$\boxed{v = \sqrt{\frac{F_0 L}{m}}}$$

**(c)** (10점) 동일한 물체가 수평면에서 일정한 힘 $F$로 끌려간다. 힘의 방향은 수평면에 대해 각도 $\phi$를 이룬다. 바닥과의 운동마찰계수가 $\mu_k$일 때, 거리 $d$를 이동한 후의 속력을 구하시오. (단, 물체는 바닥에서 떨어지지 않는다.)

**풀이:**

먼저 물체에 작용하는 힘을 분석한다:

- 당기는 힘: 수평 성분 $F\cos\phi$, 수직 성분 $F\sin\phi$ (위 방향)
- 중력: $mg$ (아래)
- 수직항력: $N$ (위)
- 운동마찰력: $f_k = \mu_k N$ (운동 반대 방향)

**수직 방향 힘의 균형** (물체가 바닥에서 떨어지지 않으므로 $a_y = 0$):

$$N + F\sin\phi - mg = 0$$

$$N = mg - F\sin\phi$$

**마찰력:**

$$f_k = \mu_k N = \mu_k(mg - F\sin\phi)$$

**수평 방향 알짜힘이 한 일:**

당기는 힘의 수평 성분이 한 일:

$$W_F = F\cos\phi \cdot d$$

마찰력이 한 일 (운동 반대 방향이므로 음):

$$W_f = -f_k \cdot d = -\mu_k(mg - F\sin\phi) \cdot d$$

알짜일:

$$W_\text{net} = F\cos\phi \cdot d - \mu_k(mg - F\sin\phi) \cdot d$$

$$= d\left[F\cos\phi - \mu_k(mg - F\sin\phi)\right]$$

일-운동에너지 정리 (정지 출발, $K_i = 0$):

$$\frac{1}{2}mv^2 = d\left[F\cos\phi - \mu_k(mg - F\sin\phi)\right]$$

$$v^2 = \frac{2d}{m}\left[F\cos\phi - \mu_k(mg - F\sin\phi)\right]$$

$$\boxed{v = \sqrt{\frac{2d}{m}\left[F\cos\phi - \mu_k(mg - F\sin\phi)\right]}}$$

## 문제 3 (25점)

질량중심과 운동량 보존에 대해 답하시오.

**(a)** (7점) 입자계의 질량중심 위치 $\vec{r}_\text{com}$을 정의하고, 이를 시간에 대해 미분하여 $M\vec{v}_\text{com} = \sum m_i \vec{v}_i$를 보이시오. 여기서 $M$은 전체 질량이다.

**풀이:**

$n$개의 입자로 이루어진 계에서 질량중심의 위치는 다음과 같이 정의된다:

$$\vec{r}_\text{com} = \frac{1}{M}\sum_{i=1}^{n} m_i \vec{r}_i$$

여기서 $M = \sum_{i=1}^{n} m_i$는 계의 전체 질량이다.

양변에 $M$을 곱하면:

$$M\vec{r}_\text{com} = \sum_{i=1}^{n} m_i \vec{r}_i$$

양변을 시간 $t$에 대해 미분한다. 각 입자의 질량 $m_i$는 상수이므로:

$$M\frac{d\vec{r}_\text{com}}{dt} = \sum_{i=1}^{n} m_i \frac{d\vec{r}_i}{dt}$$

$\frac{d\vec{r}_\text{com}}{dt} = \vec{v}_\text{com}$ (질량중심의 속도), $\frac{d\vec{r}_i}{dt} = \vec{v}_i$ (각 입자의 속도)이므로:

$$\boxed{M\vec{v}_\text{com} = \sum_{i=1}^{n} m_i \vec{v}_i}$$

우변은 계의 전체 운동량 $\vec{p}_\text{tot} = \sum m_i \vec{v}_i$이므로, $\vec{p}_\text{tot} = M\vec{v}_\text{com}$이 성립한다.

---

**(b)** (8점) 뉴턴의 제2법칙을 적용하여 $\vec{F}_\text{net,ext} = M\vec{a}_\text{com}$을 보이고, 외부 힘이 없을 때 계의 질량중심 속도가 일정함(운동량 보존)을 논하시오.

**풀이:**

(a)의 결과를 다시 시간에 대해 미분한다:

$$M\frac{d\vec{v}_\text{com}}{dt} = \sum_{i=1}^{n} m_i \frac{d\vec{v}_i}{dt}$$

$$M\vec{a}_\text{com} = \sum_{i=1}^{n} m_i \vec{a}_i$$

뉴턴의 제2법칙에 의해 $m_i \vec{a}_i = \vec{F}_i$ (입자 $i$에 작용하는 알짜힘)이므로:

$$M\vec{a}_\text{com} = \sum_{i=1}^{n} \vec{F}_i$$

각 입자에 작용하는 힘은 외부 힘과 내부 힘(입자 간 상호작용)의 합이다:

$$\sum_{i=1}^{n} \vec{F}_i = \sum_{i=1}^{n} \vec{F}_{i,\text{ext}} + \sum_{i=1}^{n} \vec{F}_{i,\text{int}}$$

뉴턴의 제3법칙에 의해, 내부 힘은 쌍을 이루어 상쇄되므로 $\sum \vec{F}_{i,\text{int}} = 0$이다. 따라서:

$$\boxed{\vec{F}_\text{net,ext} = M\vec{a}_\text{com}}$$

**외부 힘이 없을 때** ($\vec{F}_\text{net,ext} = 0$):

$$M\vec{a}_\text{com} = 0 \implies \vec{a}_\text{com} = 0$$

$$\therefore \vec{v}_\text{com} = \text{일정}$$

이는 $\vec{p}_\text{tot} = M\vec{v}_\text{com} = \text{일정}$을 의미한다. 즉, 외부 힘이 없으면 계의 전체 운동량이 보존된다. 내부 폭발이나 충돌 등으로 개별 입자의 속도가 바뀌더라도 전체 운동량(질량중심의 속도)은 변하지 않는다.

---

**(c)** (10점) 질량 $M$인 우주선이 속도 $v_0$로 운동하다가 내부 폭발에 의해 동일한 질량의 두 조각으로 분리된다. 한 조각이 원래 진행 방향에 대해 $\theta$ 각도로 속력 $v_1$으로 날아갔을 때, 나머지 조각의 속력과 방향을 $M$, $v_0$, $v_1$, $\theta$로 나타내시오.

**풀이:**

내부 폭발이므로 외부 힘이 없다. 운동량이 보존된다.

좌표 설정: 원래 진행 방향을 $x$축, 수직 방향을 $y$축으로 한다.

각 조각의 질량은 $M/2$이다. 나머지 조각의 속도 성분을 $(v_{2x}, v_{2y})$라 하자.

**$x$방향 운동량 보존:**

$$Mv_0 = \frac{M}{2}v_1\cos\theta + \frac{M}{2}v_{2x}$$

$M/2$로 나누면:

$$2v_0 = v_1\cos\theta + v_{2x}$$

$$v_{2x} = 2v_0 - v_1\cos\theta$$

**$y$방향 운동량 보존:**

초기 $y$방향 운동량은 0이다.

$$0 = \frac{M}{2}v_1\sin\theta + \frac{M}{2}v_{2y}$$

$$v_{2y} = -v_1\sin\theta$$

(음의 부호는 조각 1과 반대 방향을 의미한다.)

**나머지 조각의 속력:**

$$v_2 = \sqrt{v_{2x}^2 + v_{2y}^2} = \sqrt{(2v_0 - v_1\cos\theta)^2 + (v_1\sin\theta)^2}$$

전개하면:

$$v_2 = \sqrt{4v_0^2 - 4v_0 v_1\cos\theta + v_1^2\cos^2\theta + v_1^2\sin^2\theta}$$

$$\boxed{v_2 = \sqrt{4v_0^2 - 4v_0 v_1\cos\theta + v_1^2}}$$

**방향** (원래 진행 방향에 대한 각도 $\alpha$):

$$\tan\alpha = \frac{|v_{2y}|}{v_{2x}} = \frac{v_1\sin\theta}{2v_0 - v_1\cos\theta}$$

$$\boxed{\alpha = \arctan\left(\frac{v_1\sin\theta}{2v_0 - v_1\cos\theta}\right)}$$

이 각도는 원래 진행 방향($x$축) 기준으로, 조각 1이 날아간 방향의 **반대쪽** (아래쪽)이다.

## 문제 4 (20점) — 회전운동과 관성모멘트

**(a)** (8점) 질량 $M$, 반지름 $R$인 균일한 원판(solid disk)의 중심축에 대한 관성모멘트가 $I = \frac{1}{2}MR^2$임을 적분으로 유도하시오.

**풀이:**

원판을 반지름 $r$, 두께 $dr$인 얇은 고리(annulus)로 분할한다.

원판의 면적: $A = \pi R^2$

균일한 원판의 면밀도: $\sigma = \frac{M}{\pi R^2}$

반지름 $r$, 두께 $dr$인 고리의 면적: $dA = 2\pi r \, dr$

고리의 질량: $dm = \sigma \, dA = \frac{M}{\pi R^2} \cdot 2\pi r \, dr = \frac{2M}{R^2} r \, dr$

이 고리의 모든 질량은 축으로부터 거리 $r$에 있으므로, 고리의 관성모멘트는:

$$dI = r^2 \, dm = r^2 \cdot \frac{2M}{R^2} r \, dr = \frac{2M}{R^2} r^3 \, dr$$

전체 관성모멘트는 $r = 0$에서 $r = R$까지 적분한다:

$$I = \int_0^R \frac{2M}{R^2} r^3 \, dr = \frac{2M}{R^2} \cdot \frac{r^4}{4}\bigg|_0^R = \frac{2M}{R^2} \cdot \frac{R^4}{4}$$

$$\boxed{I = \frac{1}{2}MR^2}$$

---

**(b)** (5점) 평행축 정리(parallel-axis theorem) $I = I_\text{com} + Mh^2$을 서술하고, (a)의 원판 가장자리를 지나는 축에 대한 관성모멘트를 구하시오.

**풀이:**

**평행축 정리:** 질량중심을 지나는 축에 대한 관성모멘트가 $I_\text{com}$일 때, 이 축과 평행하고 거리 $h$만큼 떨어진 축에 대한 관성모멘트는:

$$I = I_\text{com} + Mh^2$$

원판의 가장자리를 지나는 축은 중심축과 평행하고, 거리 $h = R$만큼 떨어져 있다.

$$I_\text{edge} = I_\text{com} + MR^2 = \frac{1}{2}MR^2 + MR^2$$

$$\boxed{I_\text{edge} = \frac{3}{2}MR^2}$$

**(c)** (7점) 원판의 중심축에 줄이 감겨 있고, 줄 끝에 일정한 힘 $F$를 가한다 (축을 통해 실이 풀리는 방향). 줄의 질량은 무시한다.
- 원판의 각가속도 $\alpha$를 구하시오.
- 줄을 길이 $\ell$만큼 풀었을 때 원판의 각속도 $\omega$를 에너지 보존으로 구하시오.
- 같은 결과를 회전 운동학 식 $\omega^2 = \omega_0^2 + 2\alpha\Delta\theta$를 이용하여 확인하시오.

**풀이:**

**각가속도 $\alpha$:**

힘 $F$가 반지름 $R$에서 접선 방향으로 작용하므로, 토크는:

$$\tau = FR$$

뉴턴의 회전 제2법칙 $\tau = I\alpha$에서:

$$FR = \frac{1}{2}MR^2 \cdot \alpha$$

$$\boxed{\alpha = \frac{2F}{MR}}$$

**에너지 보존으로 $\omega$ 구하기:**

줄을 길이 $\ell$만큼 풀면, 힘 $F$가 한 일은:

$$W = F\ell$$

(힘의 방향과 줄의 이동 방향이 같으므로)

처음에 정지 상태($\omega_0 = 0$)이므로, 일-운동에너지 정리(회전):

$$W = \Delta K_\text{rot} = \frac{1}{2}I\omega^2 - 0$$

$$F\ell = \frac{1}{2} \cdot \frac{1}{2}MR^2 \cdot \omega^2 = \frac{1}{4}MR^2\omega^2$$

$$\omega^2 = \frac{4F\ell}{MR^2}$$

$$\boxed{\omega = \sqrt{\frac{4F\ell}{MR^2}} = 2\sqrt{\frac{F\ell}{MR^2}}}$$

**회전 운동학으로 확인:**

줄을 길이 $\ell$만큼 풀었을 때 원판이 회전한 각도:

$$\ell = R\Delta\theta \implies \Delta\theta = \frac{\ell}{R}$$

$\omega_0 = 0$이므로:

$$\omega^2 = \omega_0^2 + 2\alpha\Delta\theta = 0 + 2 \cdot \frac{2F}{MR} \cdot \frac{\ell}{R} = \frac{4F\ell}{MR^2}$$

$$\omega = \sqrt{\frac{4F\ell}{MR^2}} = 2\sqrt{\frac{F\ell}{MR^2}}$$

에너지 보존 결과와 일치한다. $\checkmark$

## 문제 5 (15점) — 구름운동과 각운동량

질량 $M$, 반지름 $R$인 균일한 원통(solid cylinder, $I_\text{com} = \frac{1}{2}MR^2$)이 경사각 $\theta$인 경사면을 미끄러짐 없이 굴러 내려간다.

**(a)** (8점) 에너지 보존을 이용하여, 높이 $h$에서 정지 상태로 출발한 원통의 바닥 속력을 구하시오. 같은 높이에서 미끄러져 내려오는(회전 없이) 물체의 속력과 비교하시오.

**풀이:**

미끄러짐 없이 구르므로 마찰력은 일을 하지 않는다 (접촉점의 순간 속도가 0). 역학적 에너지가 보존된다.

**에너지 보존:** 초기(높이 $h$, 정지)와 최종(바닥) 상태를 비교한다.

$$mgh = \frac{1}{2}Mv^2 + \frac{1}{2}I\omega^2$$

미끄러짐 없는 구름 조건: $v = R\omega$이므로 $\omega = v/R$.

$$Mgh = \frac{1}{2}Mv^2 + \frac{1}{2}\left(\frac{1}{2}MR^2\right)\left(\frac{v}{R}\right)^2$$

$$Mgh = \frac{1}{2}Mv^2 + \frac{1}{4}Mv^2 = \frac{3}{4}Mv^2$$

$$v^2 = \frac{4gh}{3}$$

$$\boxed{v = \sqrt{\frac{4gh}{3}}}$$

**비교:** 회전 없이 미끄러져 내려오는 경우, 모든 위치에너지가 병진 운동에너지로 전환된다:

$$Mgh = \frac{1}{2}Mv_\text{slide}^2$$

$$v_\text{slide} = \sqrt{2gh}$$

비율: $\frac{v}{v_\text{slide}} = \sqrt{\frac{4gh/3}{2gh}} = \sqrt{\frac{2}{3}} \approx 0.816$

구르는 원통의 바닥 속력은 미끄러지는 물체보다 작다. 이는 위치에너지의 일부가 회전 운동에너지로 전환되어, 병진 운동에 사용할 수 있는 에너지가 줄어들기 때문이다.

---

**(b)** (7점) 바닥에서 원통이 마찰 없는 수평면 위에 놓인 질량 $m$인 정지한 블록과 완전비탄성 충돌한다. 충돌 후 합쳐진 물체의 속력을 구하시오. (충돌 시 원통의 회전은 무시한다. 즉 충돌 직전의 병진 운동에너지만 고려한다.)

**풀이:**

(a)에서 원통의 바닥 속력: $v = \sqrt{\frac{4gh}{3}}$

완전비탄성 충돌에서는 운동량이 보존되고, 충돌 후 두 물체가 합쳐져 같은 속도로 움직인다.

**운동량 보존** (원통의 회전 무시, 병진 운동만 고려):

$$Mv = (M + m)v_f$$

$$v_f = \frac{Mv}{M + m} = \frac{M}{M + m}\sqrt{\frac{4gh}{3}}$$

$$\boxed{v_f = \frac{M}{M + m}\sqrt{\frac{4gh}{3}}}$$
